## Preset Details

### bold-signal (Dark)

- **Background:** #1a1a1a with gradient to #2d2d2d
- **Accent:** #FF5722 (vibrant orange)
- **Card fills:** Orange/coral spectrum + dark neutrals
- **Signature:** Bold colored card as focal point, large section numbers, navigation breadcrumbs
- **Font pairing:** Archivo Black (display) + Space Grotesk (body)

### electric-studio (Dark)

- **Background:** #0a0a0a (near black)
- **Accent:** #4361ee (royal blue)
- **Card fills:** White, royal blue, dark navy spectrum
- **Signature:** Two-panel vertical split, accent bar on panel edge, quote as hero element
- **Font pairing:** Manrope ExtraBold (display) + Manrope Regular (body)

### creative-voltage (Dark)

- **Background:** #0066ff (electric blue)
- **Accent:** #d4ff00 (neon yellow)
- **Card fills:** Electric blue + deep navy spectrum
- **Signature:** Electric blue + neon yellow contrast, halftone texture patterns, neon badges
- **Font pairing:** Syne Bold (display) + Space Mono Regular (body)

### dark-botanical (Dark)

- **Background:** #0f0f0f (deep black)
- **Accent:** #d4a574 (warm gold), alt: #e8b4b8 (soft pink)
- **Card fills:** Subtle warm-tinted darks
- **Signature:** Abstract soft gradient circles, warm accents (pink, gold, terracotta), italic signature typography
- **Font pairing:** Cormorant SemiBold (display) + IBM Plex Sans Light (body)

### neon-cyber (Dark)

- **Background:** #0a0f1c (dark navy)
- **Accent:** #00ffcc (cyan), alt: #ff00aa (magenta)
- **Card fills:** Deep navy/purple spectrum
- **Signature:** Neon glow effects, grid patterns, particle backgrounds
- **Font pairing:** Rajdhani Bold (display) + Inter Regular (body)

### terminal-green (Dark)

- **Background:** #0d1117 (GitHub dark)
- **Accent:** #39d353 (terminal green), alt: #58a6ff (link blue)
- **Card fills:** Dark blue-gray spectrum
- **Signature:** Scan lines, blinking cursor, code syntax styling
- **Font pairing:** JetBrains Mono Bold (display) + JetBrains Mono Regular (body)

### notebook-tabs (Light)

- **Background:** #2d2d2d (dark outer frame)
- **Accent:** #98d4bb (mint), alt: #c7b8ea (lavender)
- **Card fills:** #f8f6f1 (cream), mint, lavender, pink, sky blue, warm yellow
- **Signature:** Paper container with shadow, colorful section tabs, binder hole decorations
- **Font pairing:** Bodoni Moda Bold (display) + DM Sans Regular (body)

### pastel-geometry (Light)

- **Background:** #c8d9e6 (soft blue)
- **Accent:** #5a7c6a (forest green)
- **Card fills:** Off-white, pink, mint, purple spectrum
- **Signature:** Rounded card with soft shadow, vertical pills on right edge
- **Font pairing:** Plus Jakarta Sans Bold (display) + Plus Jakarta Sans Regular (body)

### split-pastel (Light)

- **Background:** #f5e6dc (warm peach)
- **Accent:** #c8f0d8 (mint), alt: #e4dff0 (lavender)
- **Card fills:** Warm pastels — peach, lavender, mint, yellow, pink
- **Signature:** Split background colors, playful badge pills, grid pattern overlay
- **Font pairing:** Outfit Bold (display) + Outfit Regular (body)

- **Signature:** Abstract geometric shapes, bold bordered CTA boxes, witty conversational copy

## Usage in slides.md

```markdown
<!-- style: bold-signal -->

# My Presentation Title
```

## Usage via CLI

```bash
npm run build && npx tsx src/index.ts slides.md output/ --preset=notebook-tabs
```

## DO NOT USE (Anti-patterns)

- **Display fonts:** Inter, Roboto, Arial as display/heading font (OK for body)
- **Colors:** #6366f1 (generic indigo), purple gradients on white
- **Layouts:** Everything centered, identical card grids on every slide
- **Mixing:** Never combine elements from different presets in one presentation
- **Overuse:** Avoid using the same preset for every presentation — match vibe to content
