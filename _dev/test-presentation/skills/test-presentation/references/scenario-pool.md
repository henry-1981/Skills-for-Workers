# Scenario Pool

테스트 실행 시 아래 카테고리와 변수를 랜덤 조합하여 시나리오를 생성한다.

## Categories

### 업무 보고
- **주기**: 주간, 월간, 분기
- **부서**: 개발, 마케팅, 영업, HR, 재무, 운영
- **지표**: 매출, DAU, 전환율, NPS, 이탈률, 비용절감

### 제안서
- **유형**: 신규 서비스, 예산 요청, 인력 충원, 시스템 도입, 프로세스 개선
- **대상**: 경영진, 팀장, 투자자, 파트너사
- **규모**: 소규모(1-5명), 중규모(5-20명), 대규모(20명+)

### 교육 자료
- **유형**: 온보딩, 기술 공유, 컴플라이언스, 워크숍, 세미나
- **대상 레벨**: 신입, 주니어, 시니어, 매니저, 전직원
- **분야**: 엔지니어링, 디자인, 비즈니스, 법무, 보안

### 외부 발표
- **유형**: 컨퍼런스, 고객 프레젠테이션, 파트너 미팅, 밋업, 웨비나
- **톤**: 기술적, 비즈니스, 캐주얼, 공식
- **청중 규모**: 소규모(~30명), 중규모(~100명), 대규모(300명+)

## Mode Distribution

- 자유 모드: 70% (4-Agent Pipeline)
- 프리셋 모드: 30% (랜덤 프리셋 선택)
