# Quality Scoring Rubric

NotebookLM 수준을 기준선으로 한 LLM 채점 rubric.
각 항목 0-25점, 총점 0-100점.

## 채점 지침

채점자는 아래 기준에 따라 **관찰 가능한 증거**만으로 점수를 부여한다.
주관적 인상이 아닌 구체적 요소를 근거로 제시해야 한다.

---

## 1. 핵심 메시지 전달력 (0-25)

**기준**: 청중이 발표 후 하나의 핵심 메시지를 기억할 수 있는가?

| 점수 | 기준 |
|------|------|
| 21-25 | 핵심 메시지가 명확하고, 제목/결론 슬라이드에서 일관되게 반복됨 |
| 16-20 | 핵심 메시지가 존재하나 일부 슬라이드에서 흐려짐 |
| 11-15 | 메시지가 분산되어 있고 통일된 주장이 불명확 |
| 6-10  | 정보 나열 수준, 핵심 메시지 부재 |
| 0-5   | 슬라이드 간 연결 없음 |

**관찰 항목**:
- 제목 슬라이드에 명확한 핵심 문장이 있는가
- 결론 슬라이드가 핵심 메시지를 요약하는가
- 각 슬라이드의 소제목이 핵심 메시지를 뒷받침하는가

## 2. 논리적 흐름 / 스토리라인 (0-25)

**기준**: 슬라이드 순서가 논리적이고, 청중을 설득력 있게 이끄는가?

| 점수 | 기준 |
|------|------|
| 21-25 | 명확한 도입-전개-결론 구조, 자연스러운 전환 |
| 16-20 | 전체 흐름은 있으나 일부 전환이 어색 |
| 11-15 | 구조가 있으나 순서 변경이 필요한 슬라이드 존재 |
| 6-10  | 섹션 간 연결이 약하고 점프가 잦음 |
| 0-5   | 무작위 나열 |

**관찰 항목**:
- 도입부가 청중의 관심을 유도하는가
- 각 슬라이드가 이전 슬라이드를 전제로 하는가
- 결론이 앞선 내용을 종합하는가

## 3. 시각 디자인 적절성 (0-25)

**기준**: 디자인이 메시지 전달을 돕는가, 방해하는가?

| 점수 | 기준 |
|------|------|
| 21-25 | 일관된 시각 언어, 여백 활용, 가독성 우수 |
| 16-20 | 전반적으로 양호하나 1-2장이 과밀하거나 불일치 |
| 11-15 | 기본적 가독성은 있으나 디자인 일관성 부족 |
| 6-10  | 텍스트 과다, 여백 부족, 색상 충돌 |
| 0-5   | 가독성 심각한 문제 |

**관찰 항목**:
- 슬라이드 간 색상/폰트/레이아웃 일관성
- 텍스트 양 (슬라이드당 50단어 이하 권장)
- 시각 요소(차트, 아이콘, 이미지)의 적절한 사용
- 여백과 정렬

## 4. 청중 맞춤도 (0-25)

**기준**: 시나리오에 명시된 청중과 목적에 맞는가?

| 점수 | 기준 |
|------|------|
| 21-25 | 청중 수준에 맞는 용어, 적절한 깊이, 목적 부합 |
| 16-20 | 대체로 적절하나 일부 용어/깊이가 청중과 불일치 |
| 11-15 | 범용적 내용, 청중 특화 요소 부족 |
| 6-10  | 청중과 맞지 않는 톤이나 내용 |
| 0-5   | 완전히 다른 청중을 대상으로 한 것처럼 보임 |

**관찰 항목**:
- 전문 용어 수준이 대상 청중에 적합한가
- 발표 목적(보고/제안/교육/발표)에 맞는 구조인가
- 톤(공식/캐주얼)이 상황에 맞는가
